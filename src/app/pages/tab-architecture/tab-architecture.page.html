Arquitectura del Proyecto Frontend (Aplicación Móvil) La cara visible del proyecto

La aplicación móvil es la principal interfaz para el usuario. Construida con tecnologías web modernas, ofrece una experiencia nativa y fluida.

Componentes Clave:

Ionic Framework

UI para aplicaciones híbridas.

Angular

Framework para la lógica y estado.

 Backend (Servidor) El cerebro de las operaciones

El servidor procesa las transacciones, ejecuta el modelo de IA y se comunica de forma segura con la aplicación móvil y la base de datos.

Tecnologías Utilizadas:

Python con Flask/Django

Para la API REST y la lógica de negocio.

Scikit-learn & TensorFlow

Librerías para el modelo de Machine Learning.

 Base de Datos Almacenamiento de datos

Se utiliza una base de datos NoSQL por su flexibilidad y escalabilidad, ideal para manejar grandes volúmenes de datos de transacciones.

Sistema Utilizado:

MongoDB

Base de datos NoSQL orientada a documentos.
